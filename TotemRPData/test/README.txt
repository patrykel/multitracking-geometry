1. Visualization

detailed documentation on visualization (in particular about iguana) in CMSSW framework is here:
https://twiki.cern.ch/twiki/bin/view/CMSPublic/SWGuideVisualization

To run it:

source /afs/...
cd to current directory
eval `scram runtime -sh`
iguana -p iguana_cfg.py

once IGUANA setup window popup, select "CMSSW",

ask for next event (Ctrl-N) or press button

then navigate through objects tree, selecting or deselecting
what you want to see
to go down right-click and choose "Enable children"

more here
https://twiki.cern.ch/twiki/bin/view/CMSPublic/SWGuideVisGeometryTutorial
